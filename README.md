# AI-Powered Debt Relief & Financial Recovery Platform

## 📌 Project Overview

The **AI-Powered Debt Relief & Financial Recovery Platform** is an intelligent web application designed to help borrowers effectively manage debt, analyze their financial health, and receive AI-powered settlement recommendations. The platform leverages Artificial Intelligence to generate personalized negotiation strategies, financial insights, and professional lender communication, enabling users to make informed financial recovery decisions.

---

# 🚀 Features

* 🔐 Secure User Registration & Login
* 💳 Loan Management Dashboard
* 📊 Financial Health Analysis

# 📥 Installation

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/AI-Debt-Relief-Platform.git
```

```bash
cd AI-Debt-Relief-Platform
```

---

## 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

---

## 3. Install Backend Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Install Frontend Dependencies

```bash
cd frontend
```

```bash
npm install
```

---

# 🔑 Environment Variables

Create a **.env** file inside the backend directory.

```env
GEMINI_API_KEY=your_google_gemini_api_key
SECRET_KEY=your_secret_key
DATABASE_URL=sqlite:///database.db
```

---

# ▶️ Running the Project

## Backend

```bash
uvicorn main:app --reload
```

Backend runs at

```
http://127.0.0.1:8000
```

---

## Frontend

```bash
npm run dev
```

Frontend runs at

```
http://localhost:5173
```
